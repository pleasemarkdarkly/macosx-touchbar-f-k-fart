## Prerequisites

* latest version of [Node.js](https://nodejs.org/) (at least 6.x.x)
* [MacBook Pro](http://www.apple.com/macbook-pro/) laptop with touch bar

## How to Install

    npm install
    
## How to Run

    npm start
